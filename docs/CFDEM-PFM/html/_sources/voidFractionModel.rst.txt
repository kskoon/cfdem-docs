.. index:: voidFractionModel

voidFractionModel command
=========================

Syntax
""""""

Defined in :ref:`couplingProperties <couplingProperties>`
dictionary.


.. parsed-literal::

   voidFractionModel model;

* model = name of the voidFractionModel to be applied

Examples
""""""""


.. parsed-literal::

   voidFractionModel centre;

.. note::

   This examples list might not be complete - please look for other models
   (voidFractionModel XY) in this documentation.

Description
"""""""""""

The *voidFractionModel* is the base class for models to represent the DEM
particle's volume in the CFD domain via a void fraction field.

Restrictions
""""""""""""


none

Default
"""""""

none
