.. index:: clockModel

clockModel command
==================

Syntax
""""""

Defined in :ref:`couplingProperties <couplingProperties>`
dictionary.


.. parsed-literal::

   clockModel model;

* model = name of the clockModel to be applied

Examples
""""""""


.. parsed-literal::

   clockModel standardClock;

.. note::

   This examples list might not be complete - please look for other clock
   models in this documentation.

Description
"""""""""""

The clockModel is the base class for models to examine the code/algorithm with
respect to run time.

Main parts of the clockModel classes were written by Josef Kerbl, JKU.

Restrictions
""""""""""""


none

Default
"""""""

none
